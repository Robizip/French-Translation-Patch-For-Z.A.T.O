French Translation Patch Of "Z.A.T.O. // I Love the World and Everything In It" , a game created by Ferry (Nopanamaman).

# Tutoriel FR  :
- 1 : Installer **"Z.A.T.O. // I Love the World and Everything In It"** depuis [Itch.io](https://nopanamaman.itch.io/z-a-t-o-i-love-the-world-and-everything-in-it) ou depuis [Steam](https://store.steampowered.com/app/4122860/ZATO__I_Love_the_World_and_Everything_In_It/).
- 2 : Aller dans **"Releases"** et télécharger la dernière version
- 3 : Décompressez les fichiers du patch.
- 4 : Ouvrez les fichiers de votre jeu et allez dans "game" pour y mettre le dossier "tl" du patch.
- 5 : Lancez le jeu, et profitez de la traduction en français. 

Si vous voulez retirer le patch de votre jeu, vous devrez enlever le dossier "tl" des fichiers de votre jeu.

# Tutoriel EN :
- 1 : Install **"Z.A.T.O. // I Love the World and Everything In It"** from [Itch.io](https://nopanamaman.itch.io/z-a-t-o-i-love-the-world-and-everything-in-it) or [Steam](https://store.steampowered.com/app/4122860/ZATO__I_Love_the_World_and_Everything_In_It/).
- 2 : Go in **"Releases"** and download the last version
- 3 : Extract patch files.
- 4 : Open your game files and paste the folder "tl" from the patch in "game" folder.
- 5 : Launch the game and enjoy the game with the french translation.
